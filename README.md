# Сайт Євгенії Полозенко — Porsche Ambassador

Односторінковий адаптивний сайт із 3D-моделлю Porsche 911 на головному екрані.
Модель анімована через **scroll scrubbing**: коли ви гортаєте сторінку, камера
пролітає навколо авто через 4 «розділи» (бейдж → силует → диск колеса → салон),
а текст плавно змінюється в такт скролу.

## Як запустити локально

Браузери блокують завантаження `.fbx`/текстур напряму з диска (`file://`)
через CORS. Потрібен простий локальний сервер:

```bash
# варіант 1 — Node.js
npx serve .

# варіант 2 — Python
python3 -m http.server 8080
```

Потім відкрити `http://localhost:8080` (або порт, який покаже консоль).

## Як опублікувати онлайн

Найпростіше — перетягнути папку на **Netlify Drop**
(https://app.netlify.com/drop) або залити в репозиторій на GitHub і
увімкнути **GitHub Pages**. Жодних збірок/білдів не потрібно — це чистий
HTML/CSS/JS.

## Структура проєкту

```
index.html            розмітка сторінки
css/style.css         стилі й адаптивність
js/main.js            3D-сцена (Three.js) + скрол-анімація (GSAP ScrollTrigger)
assets/models/        3D-модель (.fbx)
assets/textures/      текстури кузова, салону, коліс тощо
```

## Важливо про текстури

У вихідному пакеті модель (`.fbx`) і текстури постачаються окремо — файл
не містить прямих посилань «яка текстура на яку деталь». У `js/main.js`
є «розумний» підбір текстур за ключовими словами в назві матеріалу
(секція `BUCKETS`). Це надійно працює для типових назв (`leather`, `rim`,
`glass`, `body`, `brake`, `chrome` тощо), але якщо якась деталь після
завантаження сайту виглядає не так, як очікувалось:

1. Відкрийте сайт, натисніть F12 → вкладка Console.
2. Побачите список реальних назв матеріалів моделі (`[material] ...`).
3. У `js/main.js` знайдіть масив `BUCKETS` і додайте/скоригуйте потрібний
   рядок за цією назвою.

## Продакшн-порада

Файл моделі важить ~30 МБ, текстури — ще ~25 МБ, тож перше завантаження
сайту небистре (є прогрес-бар, поки все вантажиться). Для продакшн-версії
варто одного разу конвертувати `.fbx` у `.glb` зі стисненням Draco/Meshopt
(наприклад, через Blender: File → Export → glTF 2.0) — це зазвичай зменшує
вагу в 3–5 разів. Із такого файлу вантаження буде значно швидшим, особливо
на мобільному інтернеті.

## Технології

- **Three.js** — 3D-рендер моделі, освітлення, матеріали
- **GSAP + ScrollTrigger** — прив'язка анімації камери до скролу
- Чистий CSS (без фреймворків), Google Fonts: Unbounded + Manrope + Space Mono
- Поважає `prefers-reduced-motion` — для таких користувачів скрол-скраб
  вимикається на статичний перший кадр
